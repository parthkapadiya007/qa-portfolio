What is Quality Assurance?

Quality Assurance (QA) is a systematic process that ensures a product or service meets specified requirements and quality standards. It involves planning, execution, and review to detect and prevent defects.

Types of Testing

  • Manual Testing
  • Automation Testing
  • Functional Testing
  • Regression Testing
  • Performance Testing
  • API Testing
  • UI/UX Testing

QA Process

  1. Requirement Analysis
  2. Test Planning
  3. Test Case Design
  4. Test Execution
  5. Defect Logging
  6. Reporting & Test Closure
